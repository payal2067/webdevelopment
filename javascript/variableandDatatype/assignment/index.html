Variable and Datatype

What is a Variable?
A variable is a named container used to store data values.

4. Declaration vs Initialization
let x; // Declaration
x = 10; // Initialization

What are Data Types?
Data types define what kind of value a variable holds.
JavaScript has different types like:
Numbers
Strings
Boolean values
Objects, etc.

A. Primitive Data Types

1. Number
let age = 25;
let price = 99.99;

2. String
let name = "Payal";

3. Boolean
let isLoggeIn = true;

4. Undefined
let x;

5. Null
let data = null;

6.Symbol
let id = Symbol("id");

B. Non-Primitive Data Type

1. Object
let student = { name : "Payal", age: 24 };

2. Array
let numbers = [10, 40, 50];

3. Function
Function greet(){ console.log("Hello"); }

4. Date
let today = new Date();

5. RegExp (Regular Expression)
let pattern = /hello/;

6. Map
let map = new Map();
map.set("name", "Payal");

7. Set
let set = new Set([1, 3, 6]);
